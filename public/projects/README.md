# Project thumbnails

Auto-generated WebP files for `/projects` cards. **Do not edit manually.**

After adding or updating projects in `cv.json`, run:

```bash
pnpm run images:projects
```

Then commit the updated files here plus the `image` / `imageSource` fields in `cv.json`.

See `scripts/optimize-project-images.mjs` and `AGENTS.md` ‚Üí **Project thumbnails**.
